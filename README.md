# 🧠 Mental Health Companion Platform

A comprehensive AI-powered mental health platform that empowers users and therapists through intelligent conversation analysis, emotional insight, and gamified engagement.

## 🌟 Features

### 🧍 For Patients:
- **AI Chatbot**: 24/7 mental health assistant trained to listen empathetically and respond with cognitive behavioral insights.
- **Session Analytics**: AI-driven data analytics for each session, which provides insights to the patient on what to do
- **Gamification**: Encourages mental well-being habits with daily quests, streaks, achievements, and customizable avatars.
- **Mood Tracker**: Visual timeline of emotional states, captured through conversations and journaling.
- **VR Exposure Therapy (Optional)**: Guided immersive sessions to help treat anxiety, phobias, and PTSD.

### 🧑‍⚕️ For Therapists:
- **AI Session Insights**: Automatically analyzes patient conversations to extract:
  - Cognitive distortions
  - Key emotional triggers
  - Behavioral patterns
  - Summarized session notes
- **Speaker Attribution & Emotion Detection**: Distinguishes patient vs. therapist and classifies emotions per sentence.
- **Dashboard**: View patient progress, sentiment trends, and therapy effectiveness over time.

## 🧩 Tech Stack

### 💻 Frontend
- React (with TSX)
- TailwindCSS + ShadCN UI
- Chart.js / D3.js for data visualization

### 🧠 Backend
- Python Flask 
- MySQL for the database
- Gemini-based LLM for NLP & summarization
- Custom ML models for emotion & sentiment classification

### 🔐 Authentication
- JWT-based login/signup
- Role-based access for patients & therapists
